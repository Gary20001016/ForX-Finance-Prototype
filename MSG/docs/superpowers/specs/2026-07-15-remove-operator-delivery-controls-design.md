# 移除人工任务发送执行控件设计

## 决策

- “发送速率（每秒）”属于后台统一渠道调度、限流和供应商保护能力，不由人工任务操作者配置。
- “按 UID 去重”属于目标用户计算和受众快照生成的固定能力，不允许在发送策略阶段关闭。

## 前端调整

- 从人工消息任务“发送策略”步骤移除发送速率输入框。
- 从同一步骤移除用户去重开关。
- 删除对应表单初始值和无效前端状态。
- 保留自定义用户频控；频控负责跨任务同类消息接收上限，不替代任务内 UID 去重。

## 责任边界

- 目标用户模块输出已按 UID 去重的不可变受众快照。
- 后台发送服务根据平台统一配置执行速率控制、背压、熔断与供应商限流。
- 操作者只能配置业务发送时间、用户频控、安静时段、渠道和有效期。
