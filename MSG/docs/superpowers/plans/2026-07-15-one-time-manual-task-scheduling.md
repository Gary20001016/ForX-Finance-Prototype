# 一次性人工消息任务排期实施计划

**Goal:** 删除一次性任务无效的任务级频控，并让任务时区仅与指定时间发送关联。

## 用户指定约束

- 直接完成，不新增或运行自动化测试。

## 实施任务

1. 删除 `ManualFrequencyPolicy` 及人工任务保存字段。
2. 删除创建页的频控模式、次数、窗口输入、校验和摘要。
3. 任务时区只在指定时间发送时展示并必填。
4. 切换立即发送时清除计划时间和时区；切换指定时间时默认使用 `Asia/Shanghai`。
5. 指定时间的任务摘要包含时区。
6. 更新消息任务 PRD并标记旧频控设计已废弃。
7. 运行生产构建和差异检查，不运行测试套件。
8. 提交实现。
