# 多语言审核完成时间字段设计

## 目标

优化多语言审核详情顶部的信息字段，移除对操作者无直接价值的内容哈希，并明确区分机器翻译完成时间与人工审核完成时间。

## 页面调整

- 删除“源内容哈希”字段，不再在人工审核详情中展示 `sourceContentHash`。
- 原字段位置改为“审核完成时间”，展示 `reviewedAt`；尚未完成审核时显示“—”。
- 现有“完成时间”改名为“机翻完成时间”，继续展示 `translatedAt`。
- 操作者直接编写、无需机器翻译的内容不展示“机翻完成时间”，但仍展示“审核完成时间”。

## 数据处理

- 不删除领域模型中的 `sourceContentHash`，该字段仍可供底层幂等、审计或一致性校验使用。
- 审核通过时沿用现有 Store 逻辑写入 `reviewedAt`，本次只调整前端展示。

## 范围

仅修改多语言审核详情抽屉及其对应展示测试，不改变机器翻译任务、审核状态或后端数据结构。
